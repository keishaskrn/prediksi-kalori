# Deskripsi Proyek

Calories Burn Prediction App merupakan aplikasi berbasis Artificial Intelligence yang digunakan untuk memprediksi jumlah kalori yang terbakar berdasarkan beberapa input pengguna seperti usia, berat badan, dan durasi olahraga.

Aplikasi ini dibuat menggunakan bahasa pemrograman Python dengan framework Flask serta menerapkan algoritma Machine Learning untuk melakukan prediksi data.

---

# Algoritma yang Digunakan

- Linear Regression
- Artificial Neural Network (ANN)
- Backpropagation
- LSTM
- K-Means Clustering
